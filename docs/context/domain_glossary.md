# Domain Glossary

## full-game

A complete StarCraft II match including economy, production, scouting, combat, and win/loss outcome.

## macro

Economy, workers, production, expansion, technology, supply, and build order execution.

## micro

Low-level unit control such as focus fire, retreat, kiting, positioning, and spell use.

## opponent model

Module that estimates opponent opening, hidden tech, rush risk, army estimate, and related belief state from observations.

## hidden-state inference

Reasoning about unobserved enemy information under fog of war.

## replay learning

Learning features, labels, or policies from replay or match log data.

## opponent pool

Fixed set of opponents used for regression and ablation.

## league-lite

Small-scale opponent pool and historical-version self-play inspired by league training, without large-scale distributed RL.

## ablation

Controlled comparison that isolates the effect of a module or design choice.
