# Foundation

Foundation documents are the pre-project inputs and locked decisions that justified the project.

They are intentionally separated from `docs/context/`:

- `foundation/` is historical and mostly read-only.
- `context/` is current and expected to evolve during implementation.

Do not use a generic `reports/` folder as the only home for these documents. Survey, feasibility, and direction-lock materials are project foundations, not transient generated reports.
