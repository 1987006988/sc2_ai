# Phase 1 Demo Targets

This file separates required phase-1 demonstrations from optional showcase enhancements.

## Required Demo Targets

These support the definition of phase-1 completion:

1. A runnable full-game bot skeleton.
2. A fixed local opponent pool and repeatable evaluation loop.
3. A with/without opponent model ablation on the fixed opponent pool.

## Optional Showcase Enhancements

These are useful for presentation but are not blockers for phase-1 completion:

1. Structured strategy explanation.
2. Replay summary.

## Mainline Completion Conditions

Phase 1 mainline completion requires:

- runtime skeleton;
- evaluation loop;
- telemetry and replay metadata;
- opponent model interface and minimal implementation;
- fixed-opponent-pool ablation protocol.

## Showcase-only Conditions

The following do not block phase-1 completion:

- natural-language replay summaries;
- polished post-match coach output;
- presentation-focused explanation formatting.
