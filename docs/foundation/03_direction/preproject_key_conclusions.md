# Pre-project Key Conclusions

- Build a layered hybrid full-game bot, not an AlphaStar-like end-to-end mainline.
- Keep research prototypes, replay experiments, SMAC benchmarks, and LLM tools out of `src/sc2bot/`.
- Make opponent modeling the first distinct research feature.
- Treat replay learning as a mainline support path, not a separate production system.
- Treat SMAC / SMACv2 as micro-control research support, not the full-game project backbone.
